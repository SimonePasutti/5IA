# Tic Tac Toe 
di **Farhan Latif Gazi** `5^IA`

`Il progetto Tic Tac Toe` è il classico gioco da tavolo tris creato su misura per i dispositivi Android.

Per la realizzazione del progetto è stato utilizzato il modello MVC, nel quale la classe:
```
- TrisGame
```
estende la classe `Observable` di Java; e la classe:
```
- MainActivity
```
implementa la classe `Observer` sempre di Java.
Inoltre è presente anche la classe:
```
- Message
```
che non è altro che una classe che indica il protocollo di comunicazione tra l'Observer e l'Observable.

## Strumenti utilizzati per lo sviluppo del progetto

- `Android Studio`: per scrivere il codice.
